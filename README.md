# LAURA - FULL-STACK CHALLENGE

### Preparando o ambiente
```
$ virtualenv -p python3 env
$ source env/bin/activate
$ pip install requirements.txt
```

### Rodando o Back-end
Salvar a URI do banco no arquivo .env:

Rodar a aplicação:
```
$ python3 chatRoom/server.py
```

### Rodando o Front-end
```
$ npm i
$ npm start
```
